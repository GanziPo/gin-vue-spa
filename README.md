### 1. 基本介绍
  gin+vue+bootsrap搭建的web应用，集成登录,注册,jwt鉴权等基础功能
 
### 2. 使用说明
 ### 2.1 web端
   #### clone the project
    git clone https://github.com/Sakan-ya/gin-vue-spa.git

  #### enter the project directory
    cd web

  #### install dependency
    npm install

  #### develop
    npm run serve
    
 ### 2.2 server端
  #### enter the project directory
    cd server
    
  #### 安装go依赖包
    go list (go mod tidy)

  #### 编译
    go build
